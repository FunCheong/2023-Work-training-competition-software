#### 软件架构与外设

步进电机驱动采用简单的裸机架构，使用外设如下：
* **GPIO**：调试用途；
* **硬件UART**：调试用途；
* **硬件SPI**：通过自拟的通讯协议接收控制器的指令；
* **定时器**：使用**PWM+DMA双缓冲区**的方式控制步进脉冲的输出；
